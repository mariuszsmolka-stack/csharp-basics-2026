# C# - podstawy programowania 2026/2027

Materiały do kursu podstaw programowania w języku C#.

Kurs jest przeznaczony dla osób początkujących. Skupia się na programach konsolowych, podstawach składni C# oraz zagadnieniach przydatnych na początku nauki INF.04.

## Zakres

- pierwszy program w C#,
- wypisywanie danych w konsoli,
- zmienne i podstawowe typy danych,
- operatory,
- wejście i wyjście danych,
- instrukcje warunkowe,
- pętle,
- tablice jednowymiarowe,
- proste projekty konsolowe.
