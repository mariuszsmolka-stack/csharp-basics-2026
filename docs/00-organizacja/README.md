# Organizacja kursu

## Cel działu

Ten dział opisuje układ kursu i pomaga szybko sprawdzić, które materiały są już przygotowane.

## Jak korzystać z kursu

Lekcje są ułożone od najprostszych tematów do trudniejszych. Najlepiej czytać je po kolei i przepisywać przykłady do projektu konsolowego w Visual Studio.

## Co będzie dalej

Najpierw powstają krótkie lekcje teoretyczne z przykładami. Większe materiały praktyczne będą przygotowane w osobnych etapach.
