# if

## Cel lekcji

Nauczysz się wykonywać instrukcję tylko wtedy, gdy warunek jest spełniony.

## Krótkie wyjaśnienie

Instrukcja `if` sprawdza warunek. Jeżeli warunek ma wartość `true`, wykonywany jest kod w klamrach.

## Przykład kodu

```csharp
using System;

class Program
{
    static void Main()
    {
        int wiek = 18;

        if (wiek >= 18)
        {
            Console.WriteLine("Osoba pełnoletnia.");
        }
    }
}
```

## Najważniejsze elementy

- Warunek zapisujemy w nawiasach okrągłych.
- Kod zależny od warunku zapisujemy w klamrach.
- Warunek powinien dawać wynik `true` albo `false`.
- Do porównania używamy `==`, a nie `=`.

## Typowe błędy

- Zapis `if wiek >= 18` bez nawiasów.
- Użycie `=` zamiast `==`.
- Brak klamerek przy kilku instrukcjach.
- Średnik bezpośrednio po `if`, na przykład `if (wiek >= 18);`.

## Ćwiczenia kontrolne

1. Sprawdź, czy liczba jest dodatnia.
2. Sprawdź, czy liczba jest równa `0`.
3. Sprawdź, czy temperatura jest większa niż `30`.
